# Tasks: FR-RM-PROCESS-010 â€” Metaflow Process Audit as Required Terminal

**Input**: src/lib/metaflow/ (types, fs, miner, mine-run, opt-dag), bin/roadmap.ts, .roadmap/head.json
**Goal**: Any workflow that creates or mutates a roadmap DAG must end in a terminal audit chain that mines session transcript + latency, checks rich display invariants, checks integration rough points (including completed.json drift), and expands into fix nodes if thresholds violated.
